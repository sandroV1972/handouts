Class Poly

java.lang.Object
it.unimi.di.prog2.h09.Poly
public class Poly extends Object
Polys are immutable polynomials with integer coefficients.

A typical Poly is \( p = c_0 + c_1 x + c_2 x^2 + \cdots + c_n x^n \).
